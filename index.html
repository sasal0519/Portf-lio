Projeto em destaque

ProEng, meu primeiro projeto

O ProEng marca minha entrada prática no desenvolvimento de software. É uma suíte visual para engenharia e gestão, criada para transformar necessidades técnicas em uma ferramenta concreta, navegável e útil.

Software com assinatura própria

Desenvolvi o ProEng como um projeto autoral em Python, com proposta clara: reunir modelagem, organização visual, planejamento e estrutura técnica em um ambiente único. Além de ser meu primeiro projeto, ele sintetiza a forma como penso software: funcionalidade, identidade visual forte e utilidade prática.

O que o projeto evidencia

  • Capacidade de aprender construindo.
  • Interesse em interfaces e experiência de uso.
  • Base em Python orientada a produto técnico.
  • Conexão entre engenharia, documentação e software.

Tecnologias e repertório

  • Python como linguagem principal.
  • Linux como ambiente de trabalho.
  • Lua, LaTeX, Git e Bash no ecossistema pessoal de estudo e produção.
  • HTML e CSS para apresentação pública e portfólio.
