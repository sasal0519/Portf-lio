# Salomão Félix | Portfólio

Portfólio pessoal em HTML estático com foco em apresentação profissional, identidade visual forte e linguagem neobrutalista. O site reúne trajetória, projeto em destaque, habilidades e formas de contato em uma estrutura simples, rápida e fácil de manter.

## Visão geral

Este projeto foi construído para apresentar a atuação de Salomão Félix na interseção entre engenharia, organização, documentação e programação aplicada a problemas reais.

O layout valoriza:

- tipografia expressiva
- contrastes fortes
- blocos com personalidade visual
- animações sutis
- boa leitura em desktop e mobile

## Estrutura do projeto

```text
.
├── assets/
│   └── images/
├── index.html
└── README.md
```

## Tecnologias usadas

- HTML5
- CSS3
- JavaScript vanilla

## Seções do site

- `Hero`: apresentação principal, chamada profissional e links externos
- `Sobre`: resumo da base analítica, técnica e operacional
- `Trajetória`: experiência profissional e formação
- `Projeto em destaque`: apresentação do ProEng
- `Galeria`: imagens do projeto
- `Habilidades`: competências técnicas e profissionais
- `Contato`: canais para conexão profissional

## Como visualizar

Como o projeto é estático, basta abrir o arquivo `index.html` no navegador.

Se preferir rodar com um servidor local simples:

```bash
python3 -m http.server 8000
```

Depois acesse:

```text
http://localhost:8000
```

## Como editar

Os estilos e a estrutura principal estão centralizados em [index.html](/home/salomao_felix/professional-portfolio/index.html).

Pontos principais:

- conteúdo textual: edite o HTML das seções
- identidade visual: ajuste as variáveis em `:root`
- animações: revise as classes `.reveal` e o script no fim do arquivo
- imagens: substitua os arquivos em `assets/images/`
