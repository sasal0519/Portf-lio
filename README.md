# Professional Portfolio

Portfolio pessoal de Salomao Felix em HTML estatico.

## Conteudo

- `index.html`: pagina principal do portfolio
- `assets/images/`: imagens trazidas do projeto ProEng para a galeria e capa

## Publicacao

Este pacote foi separado do repositório principal do ProEng para poder ser publicado em um repositório proprio.

Opcoes comuns:

- GitHub Pages
- Netlify
- Vercel

## Observacao

O design segue uma direcao neobrutalista inspirada nas referencias visuais do perfil profissional e do projeto ProEng.
